# testing_kt3_parametrize
Pytest Parametrize

Проект для КТ3.

## Установка

1. Клонировать/скачать репозиторий
2. Установить зависимости:
    - `pip install -r requirements.txt`
3. Запустить тесты из корня проекта:
    - `pytest -v`

## Реализация

1. Напишите тест для функции is_even(number), которая проверяет, является ли число четным. Используйте параметризацию для проверки разных чисел.

2. Напишите тест для функции calculate_area(length, width), которая вычисляет площадь прямоугольника. Используйте параметризацию с несколькими параметрами для проверки различных комбинаций длины и ширины.

3. Напишите тест для функции classify_triangle(a, b, c), которая принимает длины трех сторон треугольника и возвращает тип треугольника: равносторонний, равнобедренный или разносторонний. Используйте параметризацию для тестирования различных комбинаций сторон.

*4. Доработайте решение первого задания КТ-2.
